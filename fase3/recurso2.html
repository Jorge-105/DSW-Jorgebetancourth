Atributos y Valores de HTML

> Los Atributos de HTML son unas palabras que se utilizan dentro de la etiqueta de apertura y asi poder controlar el comportamiento del elemento. Se reconocen varios tipos de atributos básicos: 1 Atributos requeridos necesitados por un tipo de elemento particular para que este funcine correctamente. 2 Atributos opcionales utilizados para modificar la funcionalidad por defecto de un tipo de elemento. 3 Atributos estandares soportados por muchos tipos de elementos. 4 Atributos de evento estas hacen que los tipos de elementos especifiquen giones o script para ser ejecutados bajo circunstancias concretas. Algunos atributos comunes son: El atributo id (identidad) proporciona un identificador unico en el ambito de todo el documento para un elemento. El atributo class se encarga de clasificar elementos similares para ser utilizados para propositos de presentación. El atributo title (titulo) esta se usa para unir una explicacion subtextual a un elemento. Atributo HTM. (11 marzo 2021). En Wikipedia.
